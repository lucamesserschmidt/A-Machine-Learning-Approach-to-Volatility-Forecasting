# Data

No data is included in this repository.

## Sources used (not redistributable)

| Data | Source | Notes |
|---|---|---|
| 5-minute stock bars (2009-2019, 54 tickers) | Provided by course supervisor | Proprietary; not for redistribution |
| LOBSTER-style order messages (2009-2019, 7 tickers) | Provided by course supervisor, LOBSTER-derived | Redistribution restricted by LOBSTER's licence |
| Oxford-Man Realized Library (2000-2018, 8 indices) | [Oxford-Man Institute archive](https://realized.oxford-man.ox.ac.uk/) | Public, but large; fetch directly |
| FRED macro series (VIX, EPU, T-bill) | [FRED](https://fred.stlouisfed.org/) | Public; script provided |

## Expected layout

If you have access to equivalent data, place it as:

```
data/
  stock_bars/<TICKER>_returns.RData
  lobster_messages/<TICKER>_raw_<YEAR>.RData   (or .parquet for large files)
  orderflow/<TICKER>_orderflow.csv             (generated by preprocess_lobster.py)
  oxfordmanrealizedvolatilityindices.csv
  fred_macro_predictors_1990_today.csv
```

See `src/config.py` for exact paths/patterns and `docs/DATA_RECOMMENDATIONS.md` for the full data-sourcing rationale.
